Windows boot entry missing from Dual-Boot GRUB menu | March 15th, 2022

A recent GRUB update may have removed the windows menu entry from the GRUB menu screen.

This is the result of a new security measure by not including the GRUB_DISABLE_OS_PROBER in the /etc/default/grub file by default.

Add the following flag to the grub file mentioned above:
GRUB_DISABLE_OS_PROBER=false

Then running:
grub-mkconfig -o /boot/grub/grub.cfg
should generate the updated GRUB configuration file which will now include the windows boot option in the GRUB boot screen.

Bitcoin & Crypto in general | March 9th, 2022

I really don't understand the hype behind bitcoin and how everyone is stating that this cryptocurrency has come to stay. I believe that is untrue - or so I hope.

Using bitcoin as money would be terribly unpractical and here's why (IMO):

  • A node in the blockchain has to validate whatever a block has and update the whole blockchain. This would cause an ordinary transaction (like paying groceries at the store) to take an incredible amount of time; very inefficient. With a normal bank and currency, this takes seconds.
  • If you do end up buying anything with cryptocurrency and somehow you find that you're unhappy with the item you just bought, i.e. it's damaged, you can wave goodbye to your bitcoin because, by nature, this is an irreversible action in the blockchain.
  • Crypto is praised for its annonimity however I'm confident that most people use platforms like coinbase or binance to hold their private keys. Well, these companies can now tie your identity to these private keys...
  • People are not interested in the cryptocurrency but rather on how they can profit (in non-digital currency) from buying/selling it.

Crypto in general has a terrifying carbon footprint. In fact, bitcoin alone, takes up more electricity and the whole of the Ukraine.

Oh and look, you can rent computing power to mine crypto. So much for "decentralization"...

wmenu | February 10th, 2022

Checkout this small project I made for a tiny application launcher on windows: wmenu. It is pretty much a clone of suckless' dmenu but for windows. The main reason behind creating this small tool is that I dislike using the windows basic taskbar search which sometimes doesn't suggest the programs I want to run and it also tends to suggest files - sometimes miss typing your search causes it to open a web page (very annoying).

I have tried the PowerToys Run Utility from Microsft but sadly it can be really slow when starting and I don't really care about the looks.
